## Formato PDF

Ruta pública para ver en formato gráfico de la factura necesitamos los campos `code` y `client`.  
Estos se obtendrán al momento de generar la factura en la respuesta de la emisión de la factura.  
Ejemplo para recuperar los campos desde `invoiceUrl`:

```js
const invoiceUrl = "/pdf?code=A1C44C93BY5XHTIHLCLdWewbb921&client=YIVS45VXD60F505GRB2A";
const pdfPublic = (invoiceUrl) => {
  return {
    code: invoiceUrl.slice(10, 38),
    client: invoiceUrl.slice(46),
  };
};
```

:::note
El campo `invoiceUrl` contendrá la misma longitud en caso de que requiera guardar los campos públicos de la factura para poder enviar el link de **cucu** por email externo.
:::

```http
GET https://sandbox.cucu.bo/pdf?code={code}&client={client}
```

| Parámetro | Tipo   | Requerido | Descripción                    |
|-----------|--------|-----------|--------------------------------|
| `code`    | String | Sí        | Código público de la factura   |
| `client`  | String | Sí        | Código público del cliente     |
